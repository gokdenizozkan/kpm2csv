FAQ

Is it safe?

Yes. Every operation is done locally. See the source code and learn more about the entire process of conversion here.

How does it work?

KPM offers exporting your data into a plain text file with a unique format that is not supported by any other password managers. However, we can parse the data as the format has a pattern. And convert it to another format. That is what we do here:

  • 1. Extract the data.
  • 2. Convert it to CSV.

What can I do with the resulting CSV?

You may upload it to any other password manager service or keep it safe. CSV is a common, table-like standard and most services will support some sort of way of uploading a CSV file.

Which password managers support CSVs?

Almost all password managers support CSV format. You should be fine. I, for example, used it with 1Password.
